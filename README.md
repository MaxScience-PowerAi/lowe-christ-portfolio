# Lowe Christ - AI Engineer Portfolio

A premium, minimalist, and dynamic portfolio built with Next.js, React, Tailwind CSS, and Framer Motion. Tailored for showcasing AI engineering projects, data science workflows, and full-stack development expertise.

## Getting Started

1. Install dependencies (if not already installed):
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:8080](http://localhost:8080) in your browser.

## Features

- **Next.js & React**: Modern stack for fast, server-rendered applications.
- **Tailwind CSS v4**: Utility-first CSS framework for rapid and precise styling.
- **Framer Motion**: Smooth page transitions, scroll interactions, and animated skill badges.
- **AI-Themed Aesthetics**: Deep black backgrounds with cyan and violet glowing accents.
- **Responsive Layout**: Optimized for desktop, tablet, and mobile viewing.
- **i18n ready**: Prepared for internationalization (English/French).

## Tech Stack

- **Framework**: Next.js (App Router setup compatible)
- **Styling**: Tailwind CSS, shadcn/ui components
- **Animations**: Framer Motion, GSAP
- **Forms**: React Hook Form, Zod

## Deployment

This portfolio is easily deployable on Vercel:
1. Push your code to a GitHub repository.
2. Link the repository to your Vercel account.
3. Vercel will automatically build and deploy your site.
